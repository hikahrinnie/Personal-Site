Pomodoro Timer

Features
This app uses the Pomodoro Technique which, in short, is a time managment technique where the user will work on a task typically, for 25 minutes followed by a short break. The timer alternated between a these two periods until the task is completed. This app will allow for the user to enter in the time they estimate to be able to complete the task and customize what kind of notifications they receive as well as the color scheme.
